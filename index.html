Javascript & CSS3 Deck of Cards

The below isn't a functioning game, it is simply a demo of a deck of cards built with HTML using some CSS3 properties and one sprite image for the face cards. It renders for IE as well sans the rounded corners and flipped symbols and characters. The deck functionality is handled on the client side with javascript (jQuery). Just do a view source to grab the JS class files and the cards.css file with the "house" styling rules.

Credit for the face card images goes to: www.jfitz.com/cards

The grid based layout for the cards builds on brilliant work from Mike Hall at BrainJar.com
